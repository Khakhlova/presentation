CANVAS

<canvas> is an HTML element which can be used to draw graphics via scripting (usually JavaScript).

It can be used for rendering graphs, game graphics, fancy text, or other images.

HISTORY

In 2004 Canvas was initially introduced by Apple for use in their own Mac OS X WebKit component. It was intended for use in applications such as the Dashboard widget and the Safari browser.

Later, canvas was implemented in browsers and standardized by the Web Hypertext Application Technology Working Group (WHATWG) on new proposed specifications for next generation web technologies.

BROWSER SUPPORT

<canvas> element

In HTML, a <canvas> element looks like this:

							
						      <canvas id="myCanvas" width="200" height="100"></canvas>

The <canvas> element must have an id attribute so it can be referred to by JavaScript.

The width and height attribute is necessary to define the size of the canvas.

By default the canvas is initially 300 pixels wide and 150 pixels high.

By default the <canvas> element has no border and no content.

To see the area and borders of the <canvas> add a border style

										 
											<canvas id="myCanvas" width="200" height="100" style="border:1px solid #000000;"></canvas>

HTML Canvas Drawing

Find the <canvas> element.

This is done by using the HTML DOM method getElementById(): 

							
								const canvas = document.getElementById("myCanvas");

Create a drawing object for the canvas.

The getContext() is a built-in HTML object, with properties and methods for drawing:

							
								const ctx = canvas.getContext("2d");

Set the fillStyle of the drawing object to the color green: 

							
								ctx.fillStyle = "#00FF00";

The fillStyle property can be a CSS color, a gradient, or a pattern. The default fillStyle is black.

The fillRect(x,y,width,height) method draws a rectangle, filled with the fillStyle, on the canvas:

							
								ctx.fillRect(0, 0, 150, 75);
Example
									
										<script>
										  const canvas = document.getElementById("myCanvas");
										  const ctx = canvas.getContext("2d");
										  ctx.fillStyle = "#00FF00";
										  ctx.fillRect(0, 0, 150, 75);
										</script>

HTML Canvas Coordinates

The HTML canvas is
a two-dimensional grid.

The upper-left corner of the canvas has the coordinates (0,0)

The method used earlier - fillRect(0,0,150,75) - means:

Start at the upper-left corner (0,0) and
draw a 150x75 pixels rectangle.

Draw a Line

To draw a straight line on a canvas, the following methods are used:

  • moveTo(x,y) - defines the starting point of the line
  • lineTo(x,y) - defines the ending point of the line

To actually draw the line, the following method is used:
stroke()

									
										const canvas = document.getElementById("myCanvas");
										const ctx = canvas.getContext("2d");
										ctx.moveTo(0, 0);
										ctx.lineTo(100, 50);
										ctx.strokeStyle = '#00FF00';
										ctx.stroke();

Draw a Circle

To draw a circle on a canvas, the following methods are used:

  • beginPath() - begins a path
  • arc(x,y,r,startangle,endangle) - creates an arc/curve.
    To create a circle with arc():
    - Set start angle to 0 and end angle to 2*Math.PI.
    - The x and y parameters define the x- and y-coordinates of the center of the circle.
    - The r parameter defines the radius of the circle.

									 
										 const canvas = document.getElementById("myCanvas");
										 const ctx = canvas.getContext("2d");
										 ctx.beginPath();
										 ctx.arc(95, 50, 40, 0, 2 * Math.PI);
										 ctx.strokeStyle = '#00FF00';
										 ctx.stroke();

HTML Canvas Gradients

Gradients can be used to fill rectangles, circles, lines, text, etc.

There are two different types of gradients:

  • createLinearGradient(x,y,x1,y1) - creates a linear gradient
  • createRadialGradient(x,y,r,x1,y1,r1) - creates a radial/circular gradient

Then add two or more color stops.

The addColorStop() method specifies the color stops, and its position along the gradient.
Gradient positions can be anywhere between 0 to 1.

To use the gradient, set the fillStyle or strokeStyle property to the gradient, then draw the shape (rectangle, text, or a line).

Using createLinearGradient(x,y,x1,y1):

x X-coordinate of the starting point of the gradient
y Y-coordinate of the starting point of the gradient
x1 X-coordinate of the ending point of the gradient
y1 Y-coordinate of the ending point of the gradient 
								
									const canvas = document.getElementById("myCanvas");
									const ctx = canvas.getContext("2d");
									// Create gradient
									const grd = ctx.createLinearGradient(0, 0, 170, 0);
									grd.addColorStop(0, "#00FF00");
									grd.addColorStop(1, "#FFFFFF");
									// Fill with gradient
									ctx.fillStyle = grd;
									ctx.fillRect(10, 10, 150, 80);

Using createRadialGradient(x,y,r,x1,y1,r1):

x X-coordinate of the starting circle of the gradient
y Y-coordinate of the starting circle of the gradient
r radius of the starting circle of the gradient
x1 X-coordinate of the ending circle of the gradient
y1 Y-coordinate of the ending circle of the gradient
r1 radius of the ending circle of the gradient 
									
										const canvas = document.getElementById("myCanvas");
										const ctx = canvas.getContext("2d");
										// Create gradient
										const grd = ctx.createRadialGradient(75, 50, 5, 90, 60, 100);
										grd.addColorStop(0, "#00FF00");
										grd.addColorStop(1, "#FFFFFF");
										// Fill with gradient
										ctx.fillStyle = grd;
										ctx.fillRect(10, 10, 150, 80);

HTML Canvas Text

To draw text on a canvas, the most important property and methods are:

  • font - defines the font properties for the text
  • fillText(text,x,y) - draws "filled" text on the canvas
  • strokeText(text,x,y) - draws text on the canvas (no fill)

Using fillText()

									
									  const canvas = document.getElementById("myCanvas");
									  const ctx = canvas.getContext("2d");
									  ctx.font = "35px Arial";
									  ctx.fillText("Hello World", 20, 50);

Using strokeText()

									
									  const canvas = document.getElementById("myCanvas");
									  const ctx = canvas.getContext("2d");
									  ctx.font = "35px Arial";
									  ctx.strokeText("Hello World", 20, 50);

Add Color and Center Text

									
									  const canvas = document.getElementById("myCanvas");
									  const ctx = canvas.getContext("2d");
									  ctx.font = "35px Arial";
									  ctx.fillStyle = "#00FF00";
									  ctx.textAlign = "center";
									  ctx.fillText("Hello World", canvas.width/2, canvas.height/2);

HTML Canvas Images

To draw an image on a canvas, use the following method:

drawImage(image,x,y, width, height):

image specifies the image, video, or <canvas> element to use
x X-coordinate on the canvas where the image will be placed
y Y-coordinate on the canvas where the image will be placed
width applied image width (optional parameter)
height applied image height (optional parameter)
HTML: 
										
										  <canvas id="myCanvas" width="210" 
											  height="280" style="border:1px solid #000000;"></canvas>
										  <img id="nature" width="180" height="80" 
											  src="nature.jpg" alt="nature">
JavaScript: 
										
											window.onload = function() {
											  const canvas = document.getElementById("myCanvas");
											  const ctx = canvas.getContext("2d");
											  const img = document.getElementById("nature");
											  ctx.drawImage(img, 10, 10);
											};

SUMMARY

1. <canvas> is an HTML element

2. <canvas> is used to draw graphics, on the fly, via scripting (usually JavaScript)

3. <canvas> was introduced by Apple in 2004, later it was implemented in browsers

4. Elder versions of browsers don't support <canvas>

5. The <canvas> element must have an id attribute so it can be referred to by JavaScript

6. By default <canvas> is 300 pixels wide and 150 pixels high

7. By default <canvas> has no border and no content

8. <canvas> could be used to draw lines, circles, curves, to add text, images and videos, to create gradients, charts, animation and games
